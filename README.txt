This is a project made by C1.047 for Acme AirNav Solutions, Inc. . It is intended to be help 
airports manage their resources and coordinate operations through the development of software 
solutions. The goal of this project is to develop a WIS to assist airports in managing 
logistics  for charter flights. This includes flight scheduling, booking management, crew 
management,  incident reporting and resolution, as well as aircraft maintenance.
